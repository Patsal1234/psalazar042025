GitHub

after a commit, got to GitHub and create a repo. follow the GitHub provided prompts. push your existing local repo from your computer terminal to GitHub

  • git remote ad origin https://githubusername/githubrepo.git
  • git branch -M main
  • git push -u origin main
Command 1
Add a remote repository(GitHub) to your local respository. Origin is the name of your remote repository. The web address is where you can fid the repository on GitHub
Command 2
Set the name of the branch to main. The -M is your move/rename.
Command 3
Pushes your local main branch to the remote repository named origin.
